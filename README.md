# 📚 Library Management System (React Native)

A mobile-based Library Management System built using React Native (Expo) and Redux Toolkit.
This app allows users to manage books, members, and transactions with full CRUD functionality.

---
## 🚀 Features:
- 📖 Add, update, delete, and view books
- 👤 Manage library members
- 🔄 Track transactions (issue/return)
- 📊 Dashboard with basic statistics
- 🧭 Multi-screen navigation (Stack + Tabs)
- ⚡ State management using Redux Toolkit
- 🌐 API integration (Sheet.best / custom API)
- 📱 Clean and minimal UI design

---
# 🛠️ Tech Stack
- React Native (Expo)
- Redux Toolkit + React Redux
- React Navigation
- GraphQL (Apollo Client)
- React Native Paper (UI components)

## ▶️ Getting Started

Prerequisites
- Node.js installed
- Expo CLI installed

## ⚙️ Installation
```bash
npm install
Run the App
npm start

Then:

Scan QR with Expo Go (mobile)

OR run on emulator:

npm run android
npm run ios
```
